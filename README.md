# Portfolio - Nguyen Thi Kim Ngan

Đây là website portfolio cá nhân được xây dựng để giới thiệu bản thân, kỹ năng, các dự án và thông tin liên hệ. Dự án được thực hiện trong quá trình học Frontend nhằm rèn luyện kỹ năng thiết kế và phát triển giao diện web.

## Công nghệ sử dụng

- HTML
- CSS
- JavaScript

## Chức năng

- Giới thiệu bản thân
- Hiển thị kỹ năng
- Trưng bày các dự án
- Thông tin liên hệ
